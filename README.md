# Book Tracker — Трекер прочитанных книг

**Автор:** Арсланов Арслан
**Дата:** 01.05.26
**Вариант:** Book Tracker

## Описание программы

**Book Tracker** — это графическое приложение для ведения списка прочитанных книг. Программа позволяет:
- Добавлять книги с указанием названия, автора, жанра и количества страниц.
- Просматривать все добавленные книги в удобной таблице.
- Фильтровать книги по жанру и по количеству страниц (например, показать книги с количеством страниц больше заданного значения).
- Удалять ненужные записи.
- Сохранять данные в JSON-файл и автоматически загружать их при запуске.
- Проверять корректность ввода данных (пустые поля, тип данных для страниц).

## Требования для запуска

- **Python 3.7** или выше
- Стандартные библиотеки Python: `tkinter`, `json`, `os` (входят в поставку Python)
